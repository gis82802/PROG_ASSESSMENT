【題目】
請撰寫一個 Java 程式，讓使用者連續輸入一連串整數（最多 10000 個）。每當使用者輸入一個新數字，程式就必須立即計算並輸出到目前為止所有數字的中位數。

程式要求：

使用適當的資料結構來儲存輸入的數字，以便隨時進行排序和中位數的計算。

中位數的定義：

如果數字總數為奇數，中位數就是排序後位於中間的那個數字。

如果數字總數為偶數，中位數是排序後位於中間兩個數字的平均值。

每次輸入後，程式應立即印出當前的中位數。

程式應在使用者輸入 -1 時結束，但 -1 不應被視為資料的一部分。

【輸入】
5 
2 
8 
3 
-1

【輸出】
中位數: 5.0
中位數: 3.5
中位數: 5.0
中位數: 4.0
程式結束。
