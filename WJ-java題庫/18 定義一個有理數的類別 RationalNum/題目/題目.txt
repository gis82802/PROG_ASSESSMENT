請撰寫一個 Java 類別 RationalNum，用於表示有理數（分數）。此類別應具備以下功能：

建構子 (Constructor)：接受兩個整數，分別為分子和分母。

簡化分數：在進行任何運算或印出前，分數都應自動化簡至最簡形式。例如，4/6 應化簡為 2/3。

四則運算方法：

Add(RationalNum other)：執行分數相加。

Sub(RationalNum other)：執行分數相減。

Mul(RationalNum other)：執行分數相乘。

Div(RationalNum other)：執行分數相除。

字串表示：覆寫 toString() 方法，以 分子/分母 的格式回傳分數，並處理以下特殊情況：

如果分數大於等於 1，顯示為帶分數（例如 1+8/21）。

如果分數為 0，只顯示 0。

如果分母為 1，只顯示分子（例如 5/1 顯示為 5）。

測試程式：另外撰寫一個主類別 RationalNumEx，讓使用者可以輸入兩個分數，並測試 RationalNum 類別的四則運算功能。

【輸入】

2 3
5 7
【輸出】

a = 2/3
b = 5/7
a+b = 1+8/21
a-b = -1/21
a*b = 10/21
a/b = 14/15
