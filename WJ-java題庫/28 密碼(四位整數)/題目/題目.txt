【題目】
請撰寫一個 Java 程式，實現二次密碼驗證功能。

驗證規則：

程式要求使用者輸入一個四位數的整數作為第一次密碼。

接著，程式要求使用者再次輸入密碼進行確認。

如果兩次輸入的密碼不相同，則視為驗證失敗。

如果使用者連續三次驗證失敗，程式應拋出一個自訂的例外 IncorrectPasswordException，並終止程式執行。

如果任一次輸入的密碼不符合四位數整數的格式（例如：輸入字母、小於四位數或大於四位數），程式應提示使用者重新輸入，但不計入錯誤次數。

【輸入】
1234
5678
1111
1234
【輸出】
請再次輸入密碼進行確認：
密碼不符，請重新輸入。 (您還有 2 次機會)
請再次輸入密碼進行確認：
密碼不符，請重新輸入。 (您還有 1 次機會)
請再次輸入密碼進行確認：
密碼設定成功！
