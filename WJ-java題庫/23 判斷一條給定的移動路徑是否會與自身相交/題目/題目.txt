【題目】
請撰寫一個 Java 程式，判斷一條給定的移動路徑是否會與自身相交。

給定條件：

程式會接收一個字串 path，由 'N', 'S', 'E', 'W' 四個字元組成，分別代表向北、向南、向東、向西移動一個單位。

起始點為二維平面上的原點 (0, 0)。

判斷規則：

在沿著 path 走動的過程中，如果任何時候你到達了一個之前已經訪問過的位置（除了原點），則視為路徑相交。

程式最終需回傳 true 或 false。

範例說明：

path = "NES"：從 (0,0) 走到 (0,1)，再到 (1,1)，最後到 (1,0)。沒有重複路徑，因此回傳 false。

path = "NESWW"：從 (0,0) 走到 (0,1)，再到 (1,1)，然後到 (1,0)，接著回到 (0,0)。此時回到了原點，與起始點重疊，因此回傳 true。

【輸入】
NESWW
【輸出】
true
