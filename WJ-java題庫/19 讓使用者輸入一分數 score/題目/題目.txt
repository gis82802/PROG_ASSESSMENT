【題目】

請撰寫一個 Java 程式，讓使用者輸入一個介於 0 到 100 之間的分數 score。程式應根據分數範圍，印出不同的評語。

評分規則：

60 ≤ score ≤ 100：印出 "及格"

50 ≤ score ≤ 59：印出 "須補考"

0 ≤ score ≤ 49：印出 "不及格"

score < 0 或 score > 100：提示使用者輸入不合法，並要求重新輸入。

程式應使用迴圈來持續讀取輸入，直到使用者輸入有效的分數為止。

【輸入】

-10
120
75
【輸出】

分數不合理，請重新輸入。
分數不合理，請重新輸入。
及格。
