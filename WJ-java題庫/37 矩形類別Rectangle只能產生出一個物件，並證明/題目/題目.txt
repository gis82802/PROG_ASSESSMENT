【題目 】
請撰寫一個 Java 程式，建立一個名為 Rectangle 的類別，並設計使其只能被實例化一次（單例模式）。在 main 方法中，透過嘗試建立多個 Rectangle 物件來證明你做到了這點，並根據判斷結果印出相應的訊息。

類別設計：

必須有一個私有的靜態屬性來保存唯一的實例。

建構子必須是私有的，以防止外部直接使用 new 關鍵字來建立物件。

必須提供一個公有的靜態方法，用於獲取這個唯一的實例。

主程式證明：

嘗試建立兩個 Rectangle 物件 rect1 和 rect2。

比較這兩個物件的記憶體位址。如果它們是同一個物件，則印出 "成功：兩個物件指向同一個實例。"；否則，印出 "失敗：建立了多個實例。"

【輸入】
本題無任何輸入。

【輸出】
成功：兩個物件指向同一個實例。
