【題目】
請撰寫一個 Java 函式 spinMatrix(int n, int dir)，用來產生一個 n
timesn 的螺旋矩陣。

函式參數說明：

n：一個整數，代表矩陣的維度大小，範圍為 1
leqn
leq20。

dir：一個整數，代表螺旋方向。dir=1 表示順時針螺旋，dir=2 表示逆時針螺旋。

程式要求：

在 main 方法中，讓使用者輸入測試案例的數量。

對於每個測試案例，讀取 n 和 dir 的值。

呼叫 spinMatrix 函式，並將回傳的螺旋矩陣印出。
