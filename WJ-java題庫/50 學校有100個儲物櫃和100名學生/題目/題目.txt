【題目】
請撰寫一個 Java 程式，模擬一個有 100 個儲物櫃和 100 名學生的情境。

情境規則：

有 100 個儲物櫃，編號從 1 到 100。初始狀態都為關閉。

有 100 名學生，編號從 S1 到 S100。

學生 S1 走過所有儲物櫃，並改變每個儲物櫃的狀態（打開）。

學生 S2 走過第 2、4、6... 等每個第二個儲物櫃，並改變其狀態。

學生 S3 走過第 3、6、9... 等每個第三個儲物櫃，並改變其狀態。

依此類推，直到學生 S100 走過第 100 個儲物櫃，並改變其狀態。

改變狀態的定義為：如果儲物櫃是打開的，就將其關閉；如果是關閉的，就將其打開。

程式應在所有學生都完成其操作後，輸出所有 100 個儲物櫃的最終狀態。

輸出格式要求：

打開的儲物櫃以 O 表示。

關閉的儲物櫃以 C 表示。

每行印出 10 個儲物櫃的狀態，總共 10 行。

【輸入】
本題無任何輸入。

【輸出】
O C C O C C C C O C 
C C C C C O C C C C 
C C C C O C C C C C 
C C C C C O C C C C 
C C C C C C C C O C 
C C C C C C C C C C 
C C C O C C C C C C 
C C C C C C C C C C 
O C C C C C C C C C 
C C C C C C C C C O
