題目
請撰寫一個 Java 程式，在給定的三維空間點中，找出距離彼此最近的兩個點。

程式要求：

點的座標應使用一個二維陣列來表示，其中每一列代表一個點，每一列的三個元素分別為 x,y,z 座標。

程式應讓使用者輸入點的數量，以及每個點的三個座標值。

你需要設計一個計算兩點之間距離的函式。三維空間中兩點 P_1(x_1,y_1,z_1) 與 P_2(x_2,y_2,z_2) 的距離公式為：

d= √[(x₂ - x₁)² + (y₂ - y₁)² + (z₂ - z₁)²]
 

​
 
程式需遍歷所有點對，找出最小距離，並印出這兩個點的座標。

輸入
6
-1 0 3
-1 -1 -1
4 1 1
2 0.5 9
3.5 2 -1
3 1.5 3

輸出
 A (4.0, 1.0, 1.0) 與 點 B (3.5, 2.0, -1.0) 之間的最小距離為 2.29128784747792
