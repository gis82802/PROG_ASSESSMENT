### 3. 改進的程式碼

```java
/*
    Filename: "App03_1004"
    Function: 本程式會讀取三個整數，並輸出這三個數的最大值、中間值和最小值。
    Author: SHUN-YE XU
    Version: 17.0.11
    Date: 2024/10/4
*/

import java.util.Scanner;

public class App03_1004
{
    public static void main(String[] args)
    {
        int a, b, c, max, mid, min;
        Scanner scanner = new Scanner(System.in);

        System.out.print("請輸入三個整數 (以空格分隔)：");
        a = scanner.nextInt();
        b = scanner.nextInt();
        c = scanner.nextInt();

        // 計算最大值
        max = a;
        if (b > max) max = b;
        if (c > max) max = c;

        // 計算最小值
        min = a;
        if (b < min) min = b;
        if (c < min) min = c;

        // 計算中間值
        mid = a + b + c - max - min;

        System.out.println("max = " + max);
        System.out.println("mid = " + mid);
        System.out.println("min = " + min);
    }
}
```
