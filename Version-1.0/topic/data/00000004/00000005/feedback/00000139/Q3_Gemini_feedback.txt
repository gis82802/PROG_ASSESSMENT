**3. 改進的程式碼**

```java
import java.util.Scanner;

public class CompareNumbers {
    public static void main(String[] args) {
        // 程式功能: 輸入三個數字，找出其中的最大值、最小值和中間值 (不使用 if-else 語句)
        Scanner scanner = new Scanner(System.in);
        System.out.println("請輸入三個數字:");

        // 輸入三個數字
        int a = scanner.nextInt();
        int b = scanner.nextInt();
        int c = scanner.nextInt();

        // 使用 Math.max 和 Math.min 找出最大值和最小值
        int max = Math.max(a, Math.max(b, c));
        int min = Math.min(a, Math.min(b, c));

        // 計算中間值 (總和 - 最大值 - 最小值)
        int mid = a + b + c - max - min;

        // 輸出結果
        System.out.println("最大值: " + max);
        System.out.println("中間值: " + mid);
        System.out.println("最小值: " + min);

        scanner.close(); // 關閉scanner，避免資源洩漏
    }
}
```

**4. 修改說明**

*   **增加程式功能描述:** 在 `main` 函式開頭增加了程式功能的描述，方便其他開發者理解程式碼的作用。
*   **增加提示訊息:** 將`System.out.println("3 numbers:");`修改為`System.out.println("請輸入三個數字:");`，使程式碼更友善。
*   **增加scanner close:** 添加了`scanner.close();`關閉scanner，避免資源洩漏。
